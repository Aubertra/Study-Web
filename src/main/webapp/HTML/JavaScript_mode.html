状态栏

Javascript知识点框架

 

一.DOM对象

1.文档: 一个页面就是一个文档,DOM中用document表示;

2.元素: 页面中所有标签都是元素,DOM中用element表示;

3.节点: 网页中所有元素都是节点(文本,标签,属性,注释),DOM中用node表示;

 

二.BOM对象

 

三.API方法

1.addEventListner(type: keyof DocumentEventMap, function, useCapture)

1.type: keyof DocumentEventMap: 事件名称

1.鼠标事件
click: 用户单击某个对象时调用事件响应方法
contextmenu: 用户打开上下文菜单时调用事件响应方法
dbclick: 用户双击某个对象时调用事件响应方法
mousedown: 用户按下鼠标按键时调用事件响应方法
mouseup: 用户将鼠标按键松开时调用事件响应方法
mouseenter: 用户将鼠标移入元素时调用事件响应方法
mouseleave: 用户将鼠标移出元素时调用事件响应方法
mousemove: 用户移动鼠标时调用事件响应方法
mouseover: 用户将鼠标覆盖元素时调用事件响应方法
mouseout: 用户将鼠标移开元素时调用事件响应方法
wheel: 目标元素上鼠标滚轮滚动时调用事件响应方法
 
2.键盘事件
keydown: 用户按下键盘时调用事件响应方法
keypress: 用户按下并松开键盘时调用事件响应方法
keyup: 用户松开键盘时调用事件响应方法
 
3.框架/对象(frame/object)事件
abort: 图片加载失败时调用事件响应方法
beforeunload: 离开网站(刷新或关闭)时调用事件响应方法
load: 图片或网站加载成功时调用事件响应方法
unload: 用户退出界面时调用事件响应方法
error: 在加载文档或图像发生错误时调用事件响应方法
hashchange: 在当前url锚部分变化时调用事件响应方法
pageshow: 用户访问页面时调用事件响应方法
pagehide: 用户转跳离开页面时调用事件响应方法
resize: 重新调整页面大小时调用事件响应方法
 
4.表单事件
blur: 元素失去焦点时调用事件响应方法
change: 表单内容改变时调用事件响应方法
focus: 元素获取焦点时调用事件响应方法
focusin: 元素即将获取焦点时调用事件响应方法
focusout: 元素即将失去焦点时调用事件响应方法
input: 元素获取输入时调用事件响应方法
reseach: 用户在搜索域输入时调用事件响应方法
reset: 表单重置时调用事件响应方法
 
5.剪贴板事件
copy: 用户拷贝内容时调用事件响应方法
cut: 用户剪切内容时调用事件响应方法
paste: 用户粘贴内容时调用事件响应方法
 
6.打印事件
afterprint: 页面开始打印后(打印窗口关闭)时调用事件响应方法
beforeprint: 页面即将开始打印时调用事件响应方法
 
7.多媒体(media)事件
abort: 视频或音频终止加载时调用事件响应方法
canplay: 视频或音频可以播放时调用事件响应方法
canplaythrough: 视频或音频可以无缓冲播放时调用事件响应方法
durationchange: 视频或音频时长变化时调用事件响应方法
emptied: 视频或音频播放列表为空时调用事件响应方法
error: 视频或音频数据加载错误时调用事件响应方法
loadeddata: 视频或音频加载当前帧时调用事件响应方法
loadedmetadata: 视频或音频加载元数据时调用事件响应方法
loadstart: 浏览器开始搜索目标视频或音频时调用事件响应方法
pause: 视频或音频暂停时调用事件响应方法
play: 视频或音频播放时调用事件响应方法
playing: 视频或音频开始播放时调用事件响应方法
progress: 视频或音频下载时调用事件响应方法
ratechange: 视频或音频播放速度时调用事件响应方法
seeked: 用户重新定位视频或音频位置后时调用事件响应方法
seeking: 用户重新定位视频或音频位置时调用事件响应方法
stalled: 浏览器获取的视频或音频不可用时调用事件响应方法
suspend: 浏览器读取媒体数据中断时调用事件响应方法
timeupdate: 媒体当前播放位置改变时调用事件响应方法
volumechange: 媒体音量变化时调用事件响应方法
waiting: 视频或音频需要缓冲时调用事件响应方法
 
8.动画事件
animationend: CSS动画结束播放时调用事件响应方法
animationiteration: CSS动画重复播放时调用事件响应方法
animationstart: CSS动画开始播放时调用事件响应方法
 
9.过渡事件
transitionend: CSS完成过渡时调用事件响应方法
 
10.其他事件
online: 浏览器在线工作时调用事件响应方法
offline: 浏览器离线工作时调用事件响应方法
message: 从对象接收到消息时调用事件响应方法
popstate: 浏览器浏览历史发生改变时调用事件响应方法
show: 窗口显示时调用事件响应方法
storage: web storage更新时调用事件响应方法
toggle: 元素打开或关闭时调用事件响应方法
visibilitychange: 元素visibilitystate改变时调用事件响应方法

2.function: 事件响应函数

一般格式: function(event){};

箭头格式：(event)=>{};

 

3.useCapture: 事件触发类型

true: 事件在捕获阶段执行

false(默认): 事件在冒泡阶段执行
